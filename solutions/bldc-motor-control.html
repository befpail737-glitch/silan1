Efficient and Quiet Sensorless Motor Control

This solution implements a sensorless Field-Oriented Control (FOC) algorithm for Brushless DC (BLDC) motors, ideal for applications like fans, pumps, and home appliances. It is based on a Silan Intelligent Power Module (IPM) that integrates the inverter and gate drivers, and a high-performance MCU with dedicated motor control peripherals. This combination enables quiet operation, high efficiency, and smooth torque control without the need for Hall sensors.

System Block Diagram

Core Advantages

  • Sensorless FOC: Reduces system cost and complexity by eliminating Hall effect sensors.
  • Quiet Operation: Sinusoidal current control minimizes torque ripple and audible noise.
  • High Integration: The IPM simplifies the power stage design and improves reliability.
  • Fast Time-to-Market: Comes with a reference algorithm library to accelerate firmware development.
